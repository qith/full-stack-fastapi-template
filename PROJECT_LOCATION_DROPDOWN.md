# 项目属地下拉列表功能

## 功能概述

项目管理系统现在支持通过下拉列表选择项目属地，提供了更好的用户体验和数据一致性。

## 支持的属地选项

系统支持以下6个地区选项：

1. **华北** - 蓝色标识 (primary)
2. **华中** - 紫色标识 (secondary)  
3. **华南** - 绿色标识 (success)
4. **非深非莞** - 灰色标识 (default)
5. **澳门** - 青色标识 (info)
6. **香港** - 红色标识 (error)

## 功能特性

### 1. 下拉选择
- 在项目创建和编辑表单中，项目属地字段现在是一个下拉选择框
- 用户只能从预定义的6个地区中选择，确保数据一致性
- 支持表单验证，必填字段

### 2. 视觉标识
- 每个地区都有独特的颜色标识
- 在项目列表、项目详情和项目概览页面中，属地以彩色 Chip 形式显示
- 颜色编码便于快速识别不同地区

### 3. 代码结构
- 所有地区选项和颜色映射都定义在 `src/constants/projectConstants.ts` 中
- 使用辅助函数 `getLocationColor()` 获取对应颜色
- 便于维护和扩展

## 使用方式

### 创建项目
1. 点击"添加项目"按钮
2. 在项目属地字段中，点击下拉箭头
3. 从6个预定义地区中选择一个
4. 填写其他必填信息并保存

### 编辑项目
1. 在项目列表中找到要编辑的项目
2. 点击"编辑"按钮
3. 在项目属地字段中重新选择地区
4. 保存更改

### 查看项目
- 项目列表：每个项目卡片显示带颜色的属地标签
- 项目详情：在基本信息标签页中显示属地信息
- 项目概览：在项目卡片中显示属地标签

## 技术实现

### 常量定义
```typescript
export const PROJECT_LOCATIONS = [
  '华北', '华东', '华中', '华南', '西南', '西北', '非深非莞', '澳门', '香港', '海外'
] as const

export const LOCATION_COLORS = {
  '华北': 'primary',
  '华中': 'secondary',
  '华南': 'success',
  '非深非莞': 'default',
  '澳门': 'info',
  '香港': 'error',
} as const
```

### 组件更新
- `ProjectForm.tsx`: 项目创建/编辑表单
- `ProjectList.tsx`: 项目列表显示
- `ProjectDetail.tsx`: 项目详情页面
- `ProjectManagement.tsx`: 项目概览页面

## 扩展说明

如果需要添加新的地区选项：

1. 在 `src/constants/projectConstants.ts` 中的 `PROJECT_LOCATIONS` 数组添加新地区
2. 在 `LOCATION_COLORS` 对象中为新地区分配颜色
3. 系统会自动在所有相关组件中显示新选项

## 注意事项

- 所有地区选项都是预定义的，用户无法输入自定义地区
- 颜色标识在整个系统中保持一致
- 表单验证确保项目属地是必填字段
- 支持搜索功能，可以按地区名称搜索项目
